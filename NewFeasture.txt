Network Management
DNS management — track forward/reverse DNS records alongside IPs, with zone file export
DHCP lease viewer — pull active DHCP leases from your UDM SE and display alongside static assignments
Network topology map — visual diagram showing how networks/VLANs connect through your infrastructure devices
IP conflict detection — flag when two devices claim the same IP or MAC
Monitoring & Alerting
Uptime notifications — email or webhook alerts when a monitored host goes down (Pushover, Slack, Telegram, email via SMTP)
Bandwidth/traffic charts — pull port stats from UniFi devices and graph usage over time
SSL certificate tracker — monitor expiration dates for internal services and alert before they expire
Service port monitoring — beyond ping, check that specific TCP ports (HTTP, SSH, DNS) are responding
Device Management
Firmware tracking — record current firmware versions per device, flag when updates are available
Warranty/lifecycle tracking — purchase date, warranty expiration, EOL dates with upcoming-expiry dashboard widget
Device dependency map — which devices depend on which (e.g. AP depends on switch port, VM depends on host)
Configuration backup — store/version device configs (via SSH/SNMP) with change diffing
Security & Access
User authentication — login system with role-based access (admin/viewer) since currently the app is open
Firewall rule viewer — pull and display firewall rules from the UDM SE via the integration API
Open port audit — combine Nmap scan data to show which devices expose unexpected services
MAC allowlist/blocklist — flag unknown MACs that appear on the network
Quality of Life
Dashboard widgets — make the dashboard configurable (drag/drop widgets, choose what's shown)
Notifications center — in-app notification feed for scan results, uptime alerts, capacity warnings
Backup & restore — one-click database backup/restore from the UI (beyond CSV export)
API — REST API for integrating with other tools (Home Assistant, Grafana, etc.)
Multi-site support — manage multiple physical locations with separate network inventories under one instance
Dark/light theme per section — you have the toggle, but theme-aware charts would polish the Reports page
Automation
Auto-discovery scheduling per network — different scan intervals per subnet (scan DMZ hourly, LAN daily)
Webhook triggers — fire webhooks when IPs change state, devices go down, or capacity thresholds are hit
Ansible/Terraform inventory export — generate inventory files for infrastructure automation tools
Wake-on-LAN — send WOL packets to devices directly from the device detail page
UniFi-Specific
Client insights — historical connection data, roaming events, signal strength per client
PoE scheduling — show and manage PoE port schedules from the detail page
Guest network stats — track guest portal usage and active guest clients
UniFi Protect integration — list cameras, show recording status (you have the Protect API spec available)
